# NYCU UAV 大型固定翼飛控架構

本專案用於整理 **NYCU UAV 大型固定翼無人機**的飛控、板外電腦、感測器與地面通訊架構。

## 系統架構

```text
                         ┌─────────────────┐
                         │     遙控器       │
                         │      ELRS       │
                         └────────┬────────┘
                                  │
                                  │ RC 控制訊號
                                  ▼
┌──────────────┐        ┌─────────────────┐
│ GPS          │───────▶│ Pixhawk 6C Mini │
└──────────────┘        │     飛控電腦      │
                        └───────┬─────┬─────┘
                                │     │
                MAVLink / Serial│     │數傳模組
                                │     │TBD
                                ▼     ▼
                       ┌──────────────┐   ┌──────────┐
                       │ Jetson Orin  │   │  地面站   │
                       │ Nano         │   │   GCS    │
                       └──────┬───────┘   └──────────┘
                              │
                              │ LR24-F
                              ▼
                       ┌──────────────┐
                       │  地面站電腦   │
                       └──────────────┘

┌──────────────┐
│ 雲台攝影機    │──────▶ Jetson Orin Nano
└──────────────┘
```

## 硬體配置

### 飛控電腦

- **型號：** Pixhawk 6C Mini
- **用途：**
  - 執行飛行姿態控制
  - 接收遙控器指令
  - 接收 GPS 資料
  - 輸出馬達及舵機控制訊號
  - 與 Jetson Orin Nano 交換飛行資料
  - 將飛控資料傳送至地面站

### 板外電腦

- **型號：** Jetson Orin Nano
- **用途：**
  - 執行板外控制程式
  - 接收 Pixhawk 的飛行狀態與感測器資料
  - 處理雲台攝影機影像
  - 執行目標偵測、座標計算及追蹤控制
  - 透過 LR24-F 與地面站通訊

## 通訊架構

### 遙控器與飛機通訊

- **通訊系統：** ELRS
- **連接目標：** Pixhawk 6C Mini
- **用途：**
  - 手動飛行控制
  - 飛行模式切換
  - 緊急接管
  - 解鎖與安全控制

### Pixhawk 與地面站通訊

- **通訊協定：** MAVLink
- **數傳模組：** TBD
- **用途：**
  - 傳送飛行姿態
  - 傳送 GPS 座標
  - 傳送電池與系統狀態
  - 上傳任務航點
  - 飛行參數設定與監控

> **TBD：** 數傳模組尚未決定，後續需依照通訊距離、頻率、資料速率、法規及可靠性進行選擇。

### Jetson Orin Nano 與地面站通訊

- **通訊模組：** MicoAir LR24-F
- **用途：**
  - 傳送板外電腦執行狀態
  - 傳送目標座標
  - 傳送追蹤結果
  - 接收地面站指令
  - 傳送除飛控遙測以外的自訂資料

目前 LR24-F 主要作為 **Jetson Orin Nano 與地面站之間的資料鏈路**，不取代 ELRS 遙控鏈路。

## 感測器

### GPS

- 提供飛機目前經緯度與高度
- 提供地面速度與航向資訊
- 用於航點飛行與返航功能
- 主要連接至 Pixhawk 6C Mini

### 雲台攝影機

- 主要連接至 Jetson Orin Nano
- 用於影像擷取與目標偵測
- 可搭配雲台控制，使攝影機持續朝向目標
- 影像處理後可產生：
  - 目標畫面座標
  - 目標地理座標
  - 目標追蹤狀態
  - 飛機導引指令

## 電腦間通訊

Pixhawk 6C Mini 與 Jetson Orin Nano 預計使用 MAVLink 進行資料交換。

可能使用的實體連接方式包括：

- UART Serial
- USB
- Ethernet

實際連接方式尚未確定。

Jetson Orin Nano 可透過 MAVSDK、MAVROS 或其他 MAVLink 函式庫取得：

- 飛機 GPS 座標
- 姿態資訊
- 高度
- 飛行模式
- 電池狀態
- 任務狀態

Jetson Orin Nano 也可向 Pixhawk 傳送：

- Offboard 控制指令
- 目標位置
- 目標速度
- 目標航向
- 任務控制命令

## 預計資料流

### 手動飛行

```text
ELRS 遙控器
    ↓
ELRS 接收機
    ↓
Pixhawk 6C Mini
    ↓
馬達與舵機
```

### 飛控遙測

```text
Pixhawk 6C Mini
    ↓
數傳模組（TBD）
    ↓
地面站
```

### 板外控制

```text
雲台攝影機
    ↓
Jetson Orin Nano
    ↓
影像辨識與目標座標計算
    ↓
MAVLink / Offboard 指令
    ↓
Pixhawk 6C Mini
```

### 板外電腦與地面站通訊

```text
Jetson Orin Nano
    ↓
LR24-F 空中端
    ↓
LR24-F 地面端
    ↓
地面站電腦
```

## 系統分工

| 裝置 | 主要功能 |
|---|---|
| ELRS 遙控器與接收機 | 手動控制、模式切換及緊急接管 |
| Pixhawk 6C Mini | 飛行控制、姿態穩定、導航及致動器控制 |
| Jetson Orin Nano | 影像處理、目標偵測、追蹤與板外控制 |
| GPS | 提供位置、速度、高度與導航資訊 |
| 雲台攝影機 | 擷取影像並持續觀測目標 |
| 飛控數傳模組 | Pixhawk 與地面站之間的 MAVLink 遙測 |
| LR24-F | Jetson Orin Nano 與地面站之間的自訂資料通訊 |
| 地面站 | 飛行監控、任務規劃、參數設定與板外系統控制 |

## 安全設計

系統應保留 ELRS 作為獨立的手動控制與緊急接管鏈路。

即使 Jetson Orin Nano、LR24-F 或板外控制程式發生異常，Pixhawk 仍應能：

- 維持基本姿態控制
- 切回手動或穩定飛行模式
- 執行失聯保護
- 執行返航
- 停止接受異常的 Offboard 指令
